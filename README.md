📊 DecodeLabs Internship Projects

Welcome to my Data Analytics Internship Project Repository completed during my internship at DecodeLabs. This repository contains four end-to-end data analytics projects covering the complete analytics workflow from data cleaning and exploration to SQL-based analysis and data visualization.

These projects demonstrate practical skills in data preprocessing, exploratory data analysis (EDA), SQL querying, and business-oriented visualization using real-world e-commerce sales data.

---

1️⃣ Data Cleaning & Preparation 🧹

📖 Overview

Data quality is the foundation of accurate analysis. In this project, raw e-commerce sales data was cleaned and prepared for further analysis.

The cleaning process included handling missing values, standardizing data formats, validating business logic, checking duplicate records, and ensuring consistency across the dataset.

🚀 Features

- Dataset inspection and understanding
- Missing value identification and treatment
- Duplicate record detection
- Date format validation and conversion
- Text standardization and formatting
- Numerical value rounding and consistency checks
- Data validation for negative values
- Final data quality assessment
- Export of cleaned dataset

🛠️ Technologies Used

- Python
- Pandas
- Excel Data Handling

---

2️⃣ Exploratory Data Analysis (EDA) 📈

📖 Overview

This project focuses on exploring the cleaned sales dataset to uncover patterns, trends, and business insights.

Statistical analysis techniques were used to understand customer behavior, product performance, revenue generation, and sales trends.

🚀 Features

- Descriptive statistical analysis
- Distribution analysis using skewness
- Outlier detection using IQR method
- Correlation analysis
- Product performance evaluation
- Revenue analysis
- Coupon usage analysis
- Payment method analysis
- Referral source analysis
- Monthly sales trend analysis
- Key business insights generation

🛠️ Technologies Used

- Python
- Pandas
- Matplotlib
- Statistical Analysis

3️⃣ SQL Data Analysis 🗄️

📖 Overview

This project demonstrates how SQL can be used to analyze business data efficiently.

The cleaned dataset was loaded into a SQLite database, and multiple SQL queries were executed to answer business-related questions regarding revenue, customer spending, product performance, and sales trends.

🚀 Features

- SQLite database creation
- Data import from CSV to SQL database
- Revenue analysis using SQL
- Product sales performance analysis
- Average order value calculation
- Customer spending analysis
- Payment method performance analysis
- Coupon-based revenue analysis
- Referral source analysis
- Order status reporting
- Top customer identification

🛠️ Technologies Used

- Python
- SQLite
- SQL Queries
- Pandas

4️⃣ Data Visualization 📊

This project transforms analytical findings into visual insights using charts and graphs.
